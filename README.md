# Tutorial para la parte del smartclic core
## Bienvenid@ al tutorial de uso del test para el smartclic core
### Los pasos a seguir son:
- Marcar si está el módem montado o no.
- Pulsar en el botón de inicio de test
      1. Se realiza una verificación del marcaje de la opción del módem.
      2. Se va a comprobar si la placa tiene conexión a internet (ping).
      3. Se va a intentar la conexión por SSH.
      4. Se recoge la ID de la CPU del micro.
      5. Se comprueba el funcionamiento del watchdog y se realiza un reinicio del sistema si todo funciona correctamente.
      6. Se van a comprobar los leds de la placa y se va a pedir al usuario para introducir "s" o "n" por teclado al usuario.
      7. Se comprueba el correcto funcionamiento del RTC o básicamente del reloj de la placa.
      8. Se comprueba la correcta detección de las placas externas al CORE.
